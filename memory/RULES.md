# Правила работы Claude Code в этом проекте

## Перед началом каждой сессии
1. Прочитай MEMORY.md полностью
2. Запиши в лог: дата, время, что планируешь делать

## В процессе работы
1. После каждого выполненного шага обновляй статус в MEMORY.md
2. Если встретил проблему — запиши её в MEMORY.md в раздел "Проблемы"
3. Не удаляй старые записи из лога — только добавляй новые

## В конце каждой сессии
1. Запиши что сделано
2. Запиши что осталось сделать
3. Запиши любые вопросы для Claude чата

## Формат записи в лог
### Сессия [номер]
- Дата: ДД.ММ.ГГГГ ЧЧ:ММ
- Сделано: [список]
- Осталось: [список]
- Вопросы для Claude чата: [если есть]
